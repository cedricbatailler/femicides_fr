femicides_fr

A data set of femicides in France as reported by @feminicidesfr.

How does it work?

@feminicidesfr is a Twitter account conducting a census of the femicides reported in the press, in France. The account is managed by a feminist group composed of volunteers.

Every day, at midnight, a R script looks for the most recent tweets from @feminicidesfr which my contains new femicide reports. If found, the new reports are added to the data set.

Quickstart

If you want to open the data set in a Google Sheet document, just open a new Google Sheet document and paste the following in the first cell.

=importData("https://github.com/cedricbatailler/femicides_fr/raw/main/data/femicide.csv")

Or, you can manually download the data set here.
